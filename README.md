# カウントダウンタイマー

プレゼンテーション用のカウントダウンタイマーです。ゼロ到達後もマイナス方向へカウントダウンを継続し、超過時間を把握できます。

## 特徴

- 赤と黒を基調としたアラート風デザイン
- 7セグメントLED風のタイマー表示（MM:SS:mm形式）
- ゼロ到達後もマイナス方向へカウントダウン継続
- 残り10秒以下で点滅アニメーション
- 超過時は強調警告表示

## 動作環境

- macOS 12.0+
- Windows 10+

## ビルド

```bash
go build
```

## 実行

```bash
./countdown-timer
```

## 使い方

1. 起動すると時間入力画面が表示されます
2. 分と秒を入力してスタートボタンを押す（またはEnterキー）
3. カウントダウンが開始されます
4. ゼロになっても停止せず、マイナス方向へ継続します

## キーボードショートカット

| キー | 動作 |
|------|------|
| Enter | カウントダウン開始（入力画面） |
| Space | 一時停止 / 再開（カウントダウン画面） |
| Escape | 入力画面に戻る |
| H / ? | ヘルプを表示 |
